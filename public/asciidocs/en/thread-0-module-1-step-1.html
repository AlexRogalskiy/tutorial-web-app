Create your REST endpoint using Red Hat OpenShift Application Runtimes

  1. Open the Application link ( this will open in the right panel).

  2. To open the application launcher console, click Launch Your Project.

  3. From the Create application screen, add a name for your REST endpoint application and click Continue.

  4. From the Target Environment screen, select "Code locally, Build and Deploy".

  5. From the Mission list, select Health Check.

  6. From the Runtime list, select SpringBoot and click the blue arrow to continue.

  7. Authorize your GitHub provider and click the blue arrow to continue.

  8. Review and click Set Up Application. You’ll see a confirmation screen when setup is complete.
